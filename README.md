# Secure Banking Login UI

A responsive, banking-style login page built as part of a frontend skill evaluation.

## Features
- Clean, professional UI inspired by a banking login page
- Client-side form validation
- Simulated authentication using hardcoded credentials
- Subtle loading state and inline feedback messages

## Tech Stack
- HTML5
- CSS3 (Flexbox)
-  JavaScript

## Usage
Open `index.html` in a browser to view the login page.
